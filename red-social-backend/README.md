API backend para una red social desarrollada con Node.js, Express, TypeScript y SQL Server.

## 🚀 Tecnologías
- Node.js
- Express
- TypeScript
- TypeORM
- SQL Server
- JWT (autenticación)
- bcrypt (hash de contraseñas)

## 📦 Instalación
npm install

## ⚙️ Variables de entorno
Crear un archivo .env basado en .env.example

## ▶️ Ejecución en desarrollo
npm run dev

## 📁 Estructura (referencial)
src/
  controllers/
  services/
  repositories/
  entities/
  routes/
  config/

## 🔐 Autenticación
Se utiliza JWT para el manejo de sesiones.

## 🗄️ Base de datos
SQL Server usando TypeORM como ORM.
